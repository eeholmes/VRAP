VRAP Downloads

VRAP output files can be downloaded from this tab once you do the following:

  • Select the Data tab.
  • Select a source of input data.
  • Paste input data, upload a data file, or select a demo file, depending on the input data source you selected.
  • Click the Run VRAP button and wait for the simulation to finish.

If you change the input data source, paste in new data, upload a new data file, or select a new demo file, the output files will no longer be available for downloading.

For more information, select the Help tab.
