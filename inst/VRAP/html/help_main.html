• Data tab.
    • Step 1: Input data
      Select a data input method:
      • Paste .rav data
        • Copy a properly-formatted data file and paste the contents into the "Paste .rav data" box. (Make sure you remove the "Paste data here" message.)
          Format your data like the example data files, which you can download as described below in "Download VRAP demo files".
        • Edit the data in the box if you like.
        • When you're ready to process the data, click the "Read pasted data" button.
      • Upload .rav file
        • Click the "Browse..." button immediately beneath "Choose .rav file from your file system".
        • When you select a file from your local computer, it will be uploaded.
      • Choose a demo file
        • In the "Select demo file" list, choose one of the available demos.
    • Step 2: Choose number of runs per simulation
      • Select the number of runs for the simulation, or use the number of runs embedded in the .rav file.
        When you change your selection, the application will calculate a new estimate of the simulation run time.
    • Step 3: Run VRAP
      • Click the "Run VRAP" button. The application will switch to the "Results" tab and display a calculation-in-progress screen until the simulation run completes.
        To adjust parameters, run another simulation, or download output files or demos, return to the "Data" tab.
    • Download VRAP output files
      • To download the Byr, Esc, or Sum file generated during the most recent simulation run, click the appropriate button.
        Before you download, select your operating system from the "Choose OS" list, in order to download a file with the appropriate line endings for your platform.
        Downloading is only available after a simulation has been run.
    • Download VRAP demo files
      • To download a copy of one of the VRAP demo files, select the file in the "Download the example .rav files" list and click the "Download Example file" button.
